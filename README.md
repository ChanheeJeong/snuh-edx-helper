# snuh-edx-helper (SNUH EDX Helper)
서울대학교병원 신경과 전공의를 위한 근전도/자율신경검사 판독 보조 및 자동화 프로그램
SNUH Electrodiagnostic Test Interpretation Helper
Developed using Python and PyQt6

## 주요 기능
- **ANSFT / QSART**: 검사 수치 기반 자동 판독문 생성
- **EMG**: 2x2 토글 UI 기반 검사 근육 지정 및 EMR 자동 입력 매크로
- **PFT**: 화면 캡처(OCR) 기반 환자 정보 및 결과 인식, 정상치 자동 계산
- **매크로**: 단축키(Ctrl+Shift+Q 등)를 활용한 EMR 원클릭 붙여넣기
